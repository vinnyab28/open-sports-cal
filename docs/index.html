How to use
Each calendar card has 5 action buttons
Subscribe

Best for iPhone, Mac Calendar, and Outlook. Events sync automatically when the schedule changes.

Add to Google Calendar

Best for desktop Google Calendar users. On Android, tap this to copy the calendar URL, then subscribe via calendar.google.com on a computer (Other calendars + → From URL → Paste) — it syncs to your phone automatically.

Download .ics

Downloads the calendar file to import manually into any app. Use this if Subscribe doesn't open your calendar app.

Copy link

Copies the calendar URL to your clipboard for sharing or pasting into a calendar app manually.

View Schedule

Preview the full match schedule in the browser. Filter by team using the dropdown at the top of the panel.
